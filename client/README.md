# ScamShield AI 🛡️

**The behavioral firewall against social engineering scams.**

## Problem Statement

Scammers exploit human psychology through urgency, fear, authority, and manipulation tactics. Many victims lose money because they fail to recognize these warning signs in real time.

ScamShield AI is designed to train users to identify, resist, and respond appropriately to scam attempts before they become victims.

---

## Solution

ScamShield AI is an interactive behavioral training platform that simulates realistic scam scenarios. Users make decisions during multi-turn scam conversations and receive immediate feedback, vigilance scores, and educational guidance.

The goal is to build scam immunity through practice and awareness.

---

## Features

- 🎭 Interactive multi-turn scam simulations
- ⚡ Decision interruption system
- 📊 Vigilance scoring mechanism
- 🔄 Scenario-based behavioral training
- 💡 Immediate feedback after each decision
- 🔌 Modular scenario architecture for future expansion

---

## Current Scam Scenarios

### Internship Processing Fee Scam

Users learn how scammers exploit urgency and fear of missing opportunities.

### Electricity Disconnection Scam

Users learn how to verify claims before making payments under pressure.

## How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/nireeksha20/ScamShield-AI.git
```

2. Navigate to the client folder:

```bash
cd ScamShield-AI/client
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

5. Open the application in your browser using the URL shown in the terminal.
